# NutriSetu 🌿

A personalized diet and health tracker with a fresh botanical UI.

## Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Copy env file
cp .env.example .env

# 3. Seed the food database
npm run seed

# 4. Start the server
npm start
```

Then open `index.html` in your browser (or serve with a static server).

## Scripts
| Command | Description |
|---------|-------------|
| `npm start` | Start the Express server |
| `npm run dev` | Start with nodemon (auto-reload) |
| `npm run seed` | Seed food database |
| `npm test` | Run API tests (server must be running) |

## Stack
- **Frontend**: Vanilla HTML/CSS/JS, Chart.js
- **Backend**: Node.js, Express 4, MongoDB, Mongoose
- **Fonts**: Playfair Display + DM Sans
